# GreenTech - Garbage Management App

## 📌 About
GreenTech is a garbage management app that helps to manage diffrent types Garbage.
## 👥 User Types
- Garbage Spreader
- Garbage Collector
- Green Army
- Recycling Company

## 🚀 Features
- Upload separate garbage photos
- Connect to garbage collector
- Recycling tracking

## 🛠 Technologies Used
- Android Studio
- Java
- Firebase
